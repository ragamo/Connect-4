# Connect 4

A ES6 Pixi.js game.

This game provides a micro-framework for work with some Pixi libraries.

Demo: http://haack.cl/connect4/

## Install dependencies

```sh
$ npm install
```

## Build project

```sh
$ npm run build
```

## Dev mode

Runs a local static webserver with watchify on port 8000

```sh
$ npm run serve
```
